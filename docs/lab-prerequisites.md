# Lab Prerequisites

## Required Software Packages

- Software packages that must be installed to complete the labs for this course.

### Visual Studio Code

1. Navigate to **https://code.visualstudio.com/Download** and download the **version of Visual Studio Code** based on your **Operating System**.

2. Install **Visual Studio Code** and follow the on-screen prompts.

3. Take a screenshot of the successful installation of **Visual Studio Code** and save it as **vscode.png**.

### Visual Studio Code Configuration

1. Open **Visual Studio Code**, on the left sidebar click on the **Extensions icon**.

<img src="/img/common/lab-prerequisites/vscode-extension.png" alt="vscode-extension" width="50"/>

2. In the **EXTENSIONS: MARKETPLACE** text box enter **Live Preview** and click on **Install**.

<img src="/img/common/lab-prerequisites/vscode-extension-live-preview.png" alt="vscode-extension-live-preview" width="600"/>

3. Take a screenshot of the successful installation of Live Server and save it as **vscode-extension-live-preview.png**.

4. In the **EXTENSIONS: MARKETPLACE** text box enter **Live Server** and click on **Install**.

<img src="/img/common/lab-prerequisites/vscode-extension-live-server.png" alt="vscode-extension-live-server" width="600"/>

5. Take a screenshot of the successful installation of Live Server and save it as **vscode-extension-live-server.png**.

6. In the **EXTENSIONS: MARKETPLACE** text box enter **Prettier** and click on **Install**.

<img src="/img/common/lab-prerequisites/vscode-extension-prettier.png" alt="vscode-extension-prettier" width="600"/>

7. Take a screenshot of the successful installation of Prettier and save it as **vscode-extension-prettier.png**.

8. Click on **File > Preferences > Settings** and type **Format**.

<img src="/img/common/lab-prerequisites/vscode-extension-prettier-format.png" alt="vscode-extension-format" width="300"/>

9. Look for the **Default Formatter setting** and choose **Prettier - Code formatter**.

<img src="/img/common/lab-prerequisites/vscode-extension-prettier-default-formatter.png" alt="vscode-extension-prettier-default-formatter" width="600"/>

10. Take a screenshot and save it as **vscode-extension-prettier-default-formatter.png**.

11. Look for the **Format On Save setting** and make sure it is enabled.

<img src="/img/common/lab-prerequisites/vscode-extension-prettier-format-on-save.png" alt="vscode-extension-prettier-format-on-save" width="600"/>

12. Take a screenshot and save it as **vscode-extension-prettier-format-on-save.png**.

### Node.js

1. Navigate to **https://nodejs.org/en/download/prebuilt-installer** and download the version of Node.js based on your Operating System.

2. Install **Node.js** and follow the on-screen prompts.

:::danger Node.js installation tip

When installing **Node.js**, carefully read the on-screen prompts and be sure to enable the option to **Automatically install the necessary tools...**.

If you are using **Windows**, do the following steps:

- On Windows search for **PowerShell**.

- Right-click it and **Run as Administrator**.

- Type this command:

```text
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

- Press **enter**.

- When prompted confirm **Y**.

<img src="/img/common/lab-prerequisites/powershell.png" alt="powershell" width="600"/>

:::

3. Take a screenshot of the successful installation of Node.js and save it as **nodejs.png**.

### GitHub

1. Navigate to **https://github.com** and sign up for a **GitHub account**.

2. Currently you will have an option to use: **Google account**, **Apple account**, and **Any email account**.

3. Pay attention to the prompts, it may include but not limited to: **Password complexity**, **Username**, and **Country** / **Region**.

4. You also have to verify the account to complete the process.

5. Take a screenshot of the successful creation of the **GitHub account** and save it as **github.png**.

:::info

- When signing up for this service always choose the **personal** / **non-commercial** / **free account**.

:::
